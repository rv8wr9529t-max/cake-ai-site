# cake-ai-site

这是一个由 **Mountain AI** 和 **Manus** 联合呈现的 AI 驱动的网站项目。该项目旨在提供一个集成了多种 AI 功能的平台，包括但不限于聊天、图像生成和语音转录等。

## 项目简介

`cake-ai-site` 是一个现代化的全栈 Web 应用程序，它结合了前端的响应式设计和后端的强大功能，为用户提供流畅的交互体验和智能的 AI 服务。项目采用了以下核心技术栈：

*   **前端**: React, Vite, TypeScript, TailwindCSS
*   **后端**: Node.js, TypeScript, tRPC
*   **数据库**: Drizzle ORM (可能与 MySQL/TiDB 配合)
*   **AI 集成**: LLM (大型语言模型), 图像生成, 语音转录

## 主要功能

*   **AI 聊天**: 与智能 AI 进行实时对话。
*   **图像生成**: 根据文本描述生成图像。
*   **语音转录**: 将语音内容转换为文本。
*   **用户认证**: 安全的用户登录和注册系统。
*   **管理面板**: 用于管理用户和内容的后台界面。

## 快速开始

### 环境要求

请确保您的开发环境已安装以下工具：

*   Node.js (推荐 LTS 版本)
*   pnpm (包管理器)
*   Git

### 安装依赖

进入项目根目录，安装所有必要的依赖：

```bash
pnpm install
```

### 数据库配置

本项目使用 Drizzle ORM 进行数据库操作。您需要配置数据库连接字符串，并在首次运行前执行数据库迁移。

1.  复制 `.env.example` 文件并重命名为 `.env`。
2.  编辑 `.env` 文件，填入您的数据库连接信息。
3.  执行数据库迁移：

    ```bash
    pnpm drizzle-kit push:pg # 或根据您的数据库类型调整命令
    ```

### 运行项目

启动开发服务器：

```bash
pnpm dev
```

项目将在本地启动，通常可以通过 `http://localhost:5173` 访问。
